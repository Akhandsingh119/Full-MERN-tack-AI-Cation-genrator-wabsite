# 🤖 AI Caption Generator

A full-stack **AI-powered caption generation** web application built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js). Users can upload images and receive intelligent, context-aware captions generated by **Google Gemini AI**, with image storage powered by **ImageKit**.

---

## ✨ Features

- 🔐 **User Authentication** — Secure JWT-based registration and login
- 🖼️ **AI Caption Generation** — Upload an image and get smart captions via Google Gemini API
- 📁 **Image Storage** — Images hosted on ImageKit CDN
- 📜 **Caption History** — Browse all previously generated captions on a dedicated history page
- 🔔 **Notifications** — Real-time in-app notification system
- ⚡ **Rate Limiting** — Redis-backed request rate limiting to prevent API abuse
- 📱 **Responsive Design** — Fully responsive layout with a collapsible sidebar for mobile

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) `v18+`
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) account or local MongoDB instance
- [Redis](https://redis.io/) server (local or cloud)
- [Google Gemini API Key](https://aistudio.google.com/app/apikey)
- [ImageKit](https://imagekit.io/) account (public & private keys)

### 1. Clone the Repository

```bash
git clone <your-repository-url>
cd "Full MERN Stack AI Caption Generator"
```

### 2. Setup the Backend

See the full instructions in the **[Backend README](./Backend/README.md)**.

```bash
cd Backend
npm install
# Create and configure .env, then:
node server.js
```

### 3. Setup the Frontend

See the full instructions in the **[Frontend README](./Frontend/README.md)**.

```bash
cd Frontend
npm install
npm run dev
```

The frontend runs on `http://localhost:5173` and connects to the backend on `http://localhost:3000`.
